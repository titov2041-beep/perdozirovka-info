Как мы это делали: пошаговое руководство

Мы создали этот сайт с нуля, чтобы показать, как структурировать полезную информацию и сделать её доступной. Ниже — подробное объяснение каждого этапа, технологий и приёмов. Вы сможете повторить или использовать эти знания для своих проектов.

1. Строим каркас на HTML5

Сайт состоит из логических блоков: навигация (navbar), шапка (header), основной контент (main) с разделами, подвал (footer). Каждый раздел имеет уникальный id для плавной прокрутки. Используем семантические теги: <section>, <article>, <nav>.

<section id="symptoms">
  <div class="container">
    <h2>Симптомы передозировки</h2>
    <p>...</p>
  </div>
</section>

Зачем? Семантическая разметка улучшает SEO, помогает скринридерам, а разделение на блоки упрощает стилизацию и поддержку. Все тексты снабжены ссылками на источники — это добавляет доверия и соответствует принципам доказательной медицины.

2. Стилизация: от цвета до анимаций

Мы использовали современный CSS: Flexbox и CSS Grid для построения сеток, переменные цвета, градиенты и плавные анимации при наведении. Основной шрифт — Inter (подключен через Google Fonts). Иконки взяты из библиотеки Font Awesome 6.

.grid-2 {
  display: grid;
  grid-template-columns: repeat(2, 1fr);
  gap: 30px;
}
@media (max-width: 768px) {
  .grid-2 { grid-template-columns: 1fr; }
}

Почему это важно? Адаптивность (медиа-запросы) гарантирует, что сайт одинаково удобен на телефоне, планшете и ноутбуке. Карточки и кнопки получают эффект при наведении — так интерфейс становится живым и интуитивным.

3. Оживляем сайт с помощью JavaScript

Мы добавили простой, но полезный скрипт:

  • Плавная прокрутка к разделам при клике на пункты меню.
  • Предупреждение при нажатии на кнопку экстренного вызова — напоминание позвонить по номеру 112/103.
// Пример плавной прокрутки
document.querySelectorAll('.nav-links a').forEach(link => {
  link.addEventListener('click', (e) => {
    e.preventDefault();
    const target = document.querySelector(link.getAttribute('href'));
    target.scrollIntoView({ behavior: 'smooth' });
  });
});

Результат: навигация работает без рывков, пользователь не теряется. Важно, что JS не перегружен — сайт загружается мгновенно.

4. Контент и достоверность данных

Мы собрали актуальную информацию из клинических рекомендаций Минздрава РФ, международных баз PubMed, MSD Manual и официальной статистики. Каждый важный факт сопровождается ссылкой-сноской. В отдельном разделе «Источники» вынесены все ссылки с аннотациями — это делает проект прозрачным и научно обоснованным.
